## Solutions to Project Euler problems require

 * Java 7

 * [Frege 3.19.124](http://code.google.com/p/frege/) (or above)

## Usage

 * Invoke the REPL:

        ./frege

 * Compile a solution:

        ./compile src/main/frege/euler/Problem-0001.fr

 * Execute a solution:

        ./run euler.Problem0001
